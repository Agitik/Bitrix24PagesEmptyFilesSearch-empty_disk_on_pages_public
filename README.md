# Поиск пустых файлов на странице Bitrix 24
Данный тест проверяет все блки хранения файлов на странице и возвращает количество непрогружженых файлов.
***
После запуска теста необходимо вручную искать файлы, которые были не найдены. 
Чаще всего такое происходит, когда сотрудник опубликовал файл на странице, но удалил его со своего диска.
